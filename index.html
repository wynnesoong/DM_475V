UNIT 2

讀取 vs. 驗證 — 本質差異

同一個條碼，兩台讀取器可能得到截然不同的結果

條碼讀取器 Barcode Reader
  • 揭示條碼內的資料
  • 品質指標取決於讀取器自身設定
  • 不同讀取器、不同結果
  • 無標準化校正
  • 結果不可比對、不可追溯
★ 條碼驗證器 Barcode Verifier
  • 依國際標準評等條碼品質
  • 標準化光學條件與校正程序
  • 產出符合規範的合規報告
  • 結果可重現、可比對、可追溯
  • 適用於稽核、合規、品質管理
核心觀點：兩台不同的讀取器，對同一個條碼可能有截然不同的掃描成功率。驗證器透過標準化光學條件，消除了設備差異，確保結果具有公信力。
三層比較：資料驗證 vs. 製程管控 vs. 標準驗證
項目 Data Validation
資料驗證		 Process Control
製程管控		 Verification
標準驗證
目的 檢查資料格式正確性 監控列印品質趨勢 依 ISO 標準合規評等
條碼品質評估 不評估 評估（但犧牲校正與光源規範） 完整評估（嚴格校正與光源條件）
校正要求 無 無強制要求 每次使用前強制校正
光源條件 不限 不限 ISO 標準規定的精確角度
報告合規性 ✗ 不合規 ⚠ 僅供內部參考 ✓ 合約級合規

重要觀念：選擇正確的標準是驗證的第一步。標籤上的二維碼使用 ISO 15415，而直接打刻在零件上的 DPM 碼必須使用 ISO TR 29158 進行驗證——後者允許更多光源角度，以適應複雜表面。

UNIT 3.2

ISO 15416 — 一維條碼驗證

10 條掃描線 → 逐條評等 → 平均 = 總等級

驗證原理

驗證器沿條碼高度方向取 10 條掃描線，分別評等每一條，最終取平均值作為整體等級。評等採 A（4.0）至 F（0.0）五級制。

ISO 15416 評等參數
參數 說明
Minimum Reflectance
最低反射率		黑條是否足夠「暗」？確保條碼與背景有足夠對比
Edge Contrast
邊緣對比度		黑條與白色間隔之間的反射率差異
Modulation
調變度		局部對比變化是否均勻？偵測不均勻印刷
Decode
可解碼性		標準演算法能否成功解讀？（Pass / Fail）
Defects
缺陷		印刷瑕疵、髒污、多餘標記
Decodability
解碼裕度		條/空寬度與理想值的偏差程度
UNIT 3.3

ISO 15415 — 二維標籤碼驗證

8 項參數，最低等級 = 整體等級（木桶原則）

木桶原則：二維碼的整體等級取決於所有參數中的最低等級。一項不合格，整體即不合格。
ISO 15415 八大評等參數
Symbol Contrast符號對比度 — 最亮與最暗區域的差異
Modulation調變度 — 局部對比的一致性
Reflectance Margin反射裕度 — 全域閾值的容錯空間
Fixed Pattern Damage固定圖案損壞 — 定位圖案的完整性
Axial Non-uniformity軸向不均勻性 — X/Y 軸的比例偏差
Grid Non-uniformity網格不均勻性 — 模組位置偏移
Unused Error Correction未用糾錯容量 — 糾錯碼的消耗程度
Decode可解碼性 — 是否能成功讀取
UNIT 3.4

ISO/IEC TR 29158 — DPM 驗證

專為直接零件標記設計的驗證標準

與 ISO 15415 的關鍵差異

更精密的全域閾值演算法

DPM 表面特性複雜（金屬、塑膠、陶瓷），ISO TR 29158 採用更精密的全域閾值計算方式，適應低對比、不均勻表面。

Modulation 通常改善

因閾值演算法更貼近 DPM 的光學特性，同一個 DPM 碼在 TR 29158 下的 Modulation 評分通常優於 15415。

★ 擴展光源角度 — DPM 驗證的關鍵
光源角度 ISO 15415（標籤） ISO TR 29158（DPM） 適用場景
30° ✗ ✓ 高反射金屬表面、淺刻痕
45° ✓ ✓ 標準表面、一般應用
90° ✗ ✓ 深刻零件、粗糙表面、點陣打刻
為什麼重要？ DPM 打刻在金屬、塑膠、陶瓷等材質上，表面曲率、反射率、粗糙度各異。單一光源角度無法涵蓋所有情況——TR 29158 標準正是為此而生。

UNIT 7

客戶常見問題 Q&A

點擊展開查看詳細回答

產品規格類
Q1支援的條碼類型是否涵蓋全部？GS1 DataBar 有沒有？ ▼
DM475V 標籤版支援最多條碼類型：一維（UPC/EAN、Code 128、ITF-14、I25、Code 39、Code 93、Codabar）及二維（Data Matrix ECC200、QR、Micro QR、PDF417）。DPM 及 HD 版支援 UPC/EAN、Code 128、Code 39、Data Matrix、QR、Micro QR。GS1 DataBar 需確認最新韌體版本支援狀態。
Q2DPM 版本與標籤版本的視野範圍各是多少？ ▼
標籤版 80×60mm（工作距離 60mm）、DPM 版 35×29mm（工作距離 41mm 含側光）、HD 版 11×9mm。選擇版本的關鍵在於您最常驗證的條碼尺寸和類型。
Q3一維條碼太長超出視野怎麼辦？ ▼
超過 DPM 版 35mm 視野的一維碼，可選用標籤版（80mm 視野）。大型條碼的可讀性問題通常較少，業界對大碼做 verify 的需求也較低。
Q4DPM 版本為什麼視野比較小？ ▼
DPM 通常直接打刻在產品上，尺寸不會太大，因此需要更高解析度（3.75 mil）的鏡頭來精確解析，光學設計上視野與解析度為權衡關係。
Q5DPM 的 30/45/90 度光源什麼時候用？ ▼
DPM 版內建三種光源角度（30°/45°/90°），在軟體介面直接切換。不同材質（金屬、塑膠、陶瓷）和打刻方式（雷射、噴墨、針打）對光的反射特性不同。ISO TR 29158 標準特別允許多角度光源，正因 DPM 的表面特性多變。建議：每種新樣品三個角度都測試，選擇最佳成像角度作為標準條件。
Q6驗證器能否脫離支架手持使用？ ▼
不行。條碼「讀取」(Read) 可手持任意角度，但「驗證」(Verify) 按 ISO 標準必須在固定光學條件下進行——固定角度、固定距離、固定光源。這正是驗證結果具公信力的原因：條件一致，結果才可重現、可比對。DM475V 搭配可調式支架確保每次測試條件完全一致。
應用場景類
Q8支架可否客製？原廠是否提供工程圖面？ ▼
可以。原廠提供機頭完整工程圖面（含安裝孔位尺寸），客戶可依實際場景自行設計支架。關鍵要求：機頭到待測物的距離符合規格且保持穩定水平。可調式支架可容納最高 5.5 英寸組件，加延伸臂可達 17.5 英寸。
Q9雷雕 DPM 是否 45 度光源就夠用？ ▼
不一定。雷雕雖通常為平面，但材質反射率、雕刻深度、表面粗糙度都會影響最佳光源角度。選擇 DPM 版（含三種光源）可確保不受限於單一角度，服務範圍最廣。
Q10能否同時驗證視野內多個條碼？ ▼
條碼品質驗證的最佳實務是逐一驗證。每個條碼須在最佳光學條件的中心位置，邊緣可能有光學失真影響評分。逐一驗證是業界認可的標準流程。
Q11條碼解析度的極限是多少？ ▼
各版本最小 X 軸尺寸：標籤版 6 mil (0.15mm)、DPM 版 3.75 mil (0.095mm)、HD 版 1 mil (0.025mm)。超出視野或解析度的條碼可被讀取器「讀取」但無法進行符合 ISO 標準的「驗證」。
Q12超出標準驗證器範圍的特殊應用怎麼處理？ ▼
Cognex 另有工業讀取器可嵌入同一套驗證軟體（SBG），演算法完全相同但硬體光學不符 ISO 標準。適合作為「參考性驗證」，報告會明確標示非標準光學條件。正式 ISO 驗證仍需使用 DM475V。
報告客製類
Q13報告上的品牌名稱能否移除？能否客製化？ ▼
系統提供 PDF 和 HTML 兩種自動輸出格式。HTML 格式可編輯性最高，可整合至自有報告模板。品牌標示的客製化需與原廠確認軟體層面的可行性。
Q14報告自動輸出格式有哪些？ ▼
兩種：PDF（直接歸檔）和 HTML（可編輯）。針對每次驗證或僅針對問題條碼自動儲存。完整驗證資料可封存至 FTP 或雲端。
Q15報告中各項評分欄位的數值代表什麼？為什麼有些沒有數字？ ▼
每個驗證項目給出 Grade（A–F），部分項目同時顯示量化數值（如 Symbol Contrast 的百分比）。有些項目如 Decode 僅有 Pass/Fail，不產生數值故該欄位空白。這些都是 ISO 標準中明確定義的規則。
校正認證類
Q16設備校正如何處理？校正片的追溯性如何證明？ ▼
DM475V 使用 Calibration Card 進行校正。校正片使用序列化的 GS1 Certified Calibration Conformance Standards，追溯至 NIST（美國）或 NRC（加拿大）國家標準。每次使用前校正兩次（1D 和 2D 分別校正），確保量測結果準確可靠。
Q17校正片的有效期限與更換規則？ ▼
出廠 4 年或初次使用後 2 年，取較早到期者。每張校正片附有序號（如 S/N: 191271111）及校正日期，並含 1D Master（S/N: A000861）和 2D Master（S/N: 3969）的追溯序號。
Q18原廠自己出的校正證書是否有公信力？第三方追溯證明？ ▼
校正片並非 Cognex 自行製造與驗證。追溯鏈為：DM475V → Calibration Card → GS1 Certified Conformance Standards（第三方精密製造）→ NIST/NRC 國家標準。校正片證書載明追溯路徑，而 Certificate of Conformity 由 Cognex 資深品質工程師簽章，明列所有合規標準。
